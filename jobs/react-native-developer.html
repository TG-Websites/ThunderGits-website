About the Role

As a React Native Developer at ThunderGits, you will be responsible for developing and maintaining high-quality mobile applications for both Android and iOS platforms. You will collaborate closely with our design and backend teams to deliver user-friendly and performant applications that enhance user experiences.

Responsibilities

  • Develop and maintain cross-platform mobile applications using React Native.
  • Collaborate with designers and backend developers to create seamless app experiences.
  • Write clean, efficient, and reusable code following industry best practices.
  • Integrate third-party APIs and libraries into the applications.
  • Debug and resolve performance issues to ensure smooth user experiences.
  • Stay updated with the latest React Native advancements and mobile development trends.

Qualifications

  • Proven experience in mobile application development using React Native.
  • Strong understanding of JavaScript, ES6+, and TypeScript.
  • Familiarity with native mobile app development (iOS/Android).
  • Experience integrating RESTful APIs and third-party libraries.
  • Knowledge of state management libraries like Redux or Context API.
  • Strong debugging and problem-solving skills.
  • Familiarity with version control systems like Git.
